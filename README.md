# FoodSethu

FoodSethu is a Food Donation Management System that connects hostels with orphanages to reduce food waste and help people in need.

## Features

- Hostel Registration & Login
- Orphanage Registration & Login
- Secure Password Hashing
- Role-Based Access Control
- Add Food Donations
- Edit Donations
- Delete Donations
- Accept Donations
- Mark Donations as Delivered
- Donation History
- Location-Based Search
- Food Expiry Warning System
- Session Management
- Bootstrap Responsive UI

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Configure .env file

4. Run the application

```bash
python app.py
```
